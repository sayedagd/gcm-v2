Frontend Architecture

React application

The frontend is a standalone React + Vite application. It uses React Router for route shells, Zustand for local app state, and a mixed UI layer built with Mantine, custom utility components, and Tailwind-based styling.

Home Overview Backend Schema API Print export
  • Entry point: src/App.tsx
  • State management: Zustand store in src/store/index.ts
  • Routing: React Router with role-based route shells
  • Public pages: Landing, Store, Contact, Carbon
  • Internal portals: Admin, accountant, logistics, client, subcontractor, driver
Deployment note: build the frontend as a standalone static app and point it to the backend API through VITE_API_BASE_URL or runtime config.
