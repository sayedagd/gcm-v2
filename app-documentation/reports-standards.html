Reports Standards

Print and PDF layout rules

Printed dossiers, manifests, trip tickets, and delivery notes must follow a rigid layout. This keeps generated reports legally and operationally consistent.

Home Docs archive System knowledge Trips Deployment Print export
  • Use a fixed A4 page container for all print output.
  • Keep legal manifest tables strict, bordered, and print-safe.
  • Use the branded header/footer helpers for generated documents.
  • Prefer the browser print engine over external PDF tooling for report consistency.
