## Executive Summary

GCM v2 is a waste-management and environmental services platform used to run end-to-end operations for projects, clients, logistics teams, subcontractors, and administrators. It combines:

- Operational planning (projects, services, facilities, supplier rates)
- Execution tracking (trip lifecycle, driver/vehicle assignment, evidence files)
- Governance and visibility (audit logs, notifications, metrics, role-based boundaries)
- Public and commercial surfaces (landing/contact, environmental equipment store)
- Reliability and compliance controls (backup/restore, schema health, rate limits, CORS, JWT)

The current architecture is a split deployment:

- Backend: Express + PostgreSQL, deployed as a Vercel serverless function wrapper via `backend/api/[...path].js`
- Frontend: Next.js App Router (`frontend-next/`) with middleware-enforced role boundaries and migration-mode session cookies
