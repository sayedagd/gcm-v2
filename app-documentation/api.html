API Reference

Major endpoint groups

The backend exposes versioned route groups under /api/v1 for identity, operations, logistics, and system services. OpenAPI is the source of truth for contract generation.

  • OpenAPI contract file: backend/openapi/openapi.v1.json.
  • Frontend typed schema generation: frontend/src/api/generated/schema.d.ts.
  • Legacy non-versioned /api/* aliases are still available with deprecation headers during migration window.
  • Error responses follow canonical envelope shape with compatibility fields.
  • Expired auth tokens return 401 AUTH_TOKEN_EXPIRED and must trigger re-authentication.
  • Sensitive route groups apply stricter endpoint-tier limits (auth/public-write/SSE-token/AI/admin operations).
  • Error envelopes and response headers carry trace IDs for incident correlation across logs.
